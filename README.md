# Estudos das disciplinas de Programação 2 & Laboratório de Programação 2 (UFCG)

## Objetivo

Sou estudante de Ciência da Computação pela Universidade Federal de Campina Grande (UFCG) e decidi colocar meus estudos referentes às disciplinas de Programação 2 e Laboratório de Programação 2 (P2/LP2) em um repositório para manter registradas as minhas resoluções e estudos. Estou usando as provas disponibilizadas no [Tamburetei](https://tamburetei.opendevufcg.org/).

O foco das duas disciplinas é explorar o paradigma orientado a objetos e explorar boas práticas de design de software e modelagem de sistemas OO. 

## O que há por aqui?

Resoluções de provas antigas de LP2 (o foco) e também implementações de modelagens referentes a P2. Todos os códigos são escritos em Java.
